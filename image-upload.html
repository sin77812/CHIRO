🖼️ CHIRO 이미지 업로드 도구

이미지를 선택하고 최적화하세요

포트폴리오, 블로그, OG 이미지를 업로드하면 자동으로 최적화됩니다

📝 사용 방법

  1. 이미지 선택: 업로드할 이미지 파일을 선택합니다
  2. 최적화: 이미지를 웹에 최적화된 크기로 자동 리사이즈합니다
  3. 파일 저장: 최적화된 이미지를 images/ 폴더에 저장합니다
  4. URL 복사: 생성된 URL을 관리자 페이지에서 사용합니다
  5. Git 푸시: 이미지 파일을 GitHub에 푸시해야 URL이 활성화됩니다

🎯 최적화 사양

  • 포트폴리오: 1200x800px, 품질 85%, WebP 변환
  • 블로그: 800x600px, 품질 80%, WebP 변환
  • OG 이미지: 1200x630px, 품질 90%, JPG 유지
